coaligned invariants — Run the repository's invariant rule modules from .coaligned/invariants/

Usage: coaligned invariants [options]

Options:
  --seed=<string>  Print the named module's seed output (e.g. a refreshed deny-list) instead of checking

Global options:
  --help, -h  Show this help
  --json      Output findings as JSON

Examples:
  coaligned invariants
  coaligned invariants --seed ambient-deps
