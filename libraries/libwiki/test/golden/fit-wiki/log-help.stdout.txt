fit-wiki log [subcommand] — Append a decision/note/done entry to the current weekly log

Usage: fit-wiki log [subcommand] [options]

Options:
  --agent=<string>         Agent name (required; no environment fallback)
  --wiki-root=<string>     Override wiki root directory (default: wiki)
  --today=<string>         Override today's ISO date (testing)
  --surveyed=<string>      Decision: routing levels surveyed
  --chosen=<string>        Decision: chosen action
  --rationale=<string>     Decision: rationale
  --alternatives=<string>  Decision: alternatives
  --field=<string>         Note: field heading
  --body=<string>          Note: field body

Global options:
  --help, -h  Show this help
  --json      Render --help output as JSON
