fit-wiki inbox [subcommand] — Triage the agent's Message Inbox (list/ack/promote/drop)

Usage: fit-wiki inbox [subcommand] [options]

Options:
  --agent=<string>      Agent name (required; no environment fallback)
  --wiki-root=<string>  Override wiki root directory (default: wiki)
  --today=<string>      Override today's ISO date (testing)
  --index=<string>      Bullet index (0-based) for ack/promote/drop
  --owner=<string>      Owner field when promoting (default: --agent)

Global options:
  --help, -h  Show this help
  --json      Render --help output as JSON
