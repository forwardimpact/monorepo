fit-wiki release — Release a claim (or all expired claims with --expired)

Usage: fit-wiki release [options]

Options:
  --agent=<string>      Agent name (falls back to LIBEVAL_AGENT_PROFILE, then staff-engineer)
  --wiki-root=<string>  Override wiki root directory (default: wiki)
  --today=<string>      Override today's ISO date (testing)
  --target=<string>     Target to release
  --expired             Release every row past expires_at

Global options:
  --help, -h  Show this help
  --json      Render --help output as JSON
