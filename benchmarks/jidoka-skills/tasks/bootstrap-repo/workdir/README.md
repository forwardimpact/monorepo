# unitconv

A tiny command-line tool that converts between metric and imperial units.

```sh
unitconv 10 km mi     # -> 6.2137
unitconv 200 g oz     # -> 7.0548
```

Built for engineers who want a quick unit conversion at the terminal without
opening a browser or trusting a random web result. One binary, no network, no
dependencies beyond the Node.js standard library.
