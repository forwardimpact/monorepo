## Empowered Engineers: Prepare for the Day Ahead

**Trigger:** The todo list has grown long enough that scanning it every morning
to find the few items that matter wastes the first minutes of the day.

**Big Hire:** Help me see only the todos relevant to what I'm working on right
now, without deleting or reorganizing the rest. → **todo**

**Little Hire:** Help me narrow a long list to the items mentioning a project or
keyword. → **todo**

**Competes With:** scrolling the whole list; piping `todo list` into `grep`;
keeping separate lists per project.

**Forces:**

- **Push:** A flat list grows without bound and every item costs attention.
- **Pull:** A single command that shows just the matching items.
- **Habit:** Re-reading the entire list each morning.
- **Anxiety:** A filter might hide an item that actually mattered.

**Fired When:** the list is short enough that filtering adds friction; or the
engineer moves to a richer task manager.
