gemba-wiki memo — Send a cross-team memo into a teammate's Message Inbox

Usage: gemba-wiki memo [options]

Options:
  --from=<string>       Sender agent name (required; no environment fallback)
  --to=<string>         Target agent name, or "all" to broadcast (sender is skipped)
  --message=<string>    Memo text
  --wiki-root=<string>  Override wiki root directory (default: wiki)

Global options:
  --help, -h  Show this help
  --json      Render --help output as JSON
