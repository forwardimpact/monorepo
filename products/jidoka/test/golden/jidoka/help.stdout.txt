jidoka X.Y.Z — Enforce the layered instruction architecture defined in JIDOKA.md (no subcommand: run every check)

Usage: jidoka <command> [options]

Commands:
  instructions  Check L1–L6 length and checklist caps across the repo
  invariants    Run the repository's invariant rule modules from .jidoka/invariants/
  jtbd          Validate package.json .jobs entries and generated blocks

Options:
  --help, -h  Show this help
  --version   Show version
  --json      Output findings as JSON

Examples:
  jidoka
  jidoka instructions
  jidoka jtbd --fix

Documentation:
  Jidoka Instruction Architecture Standard
    https://github.com/forwardimpact/monorepo/blob/main/JIDOKA.md
  Jidoka website
    https://www.jidoka.team/

Use jidoka <command> --help for command-specific options.
